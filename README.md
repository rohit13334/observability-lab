# Observability Lab

This repository is a hands-on lab to learn and compare modern observability tools.

## Goals

- Instrument Spring Boot applications using OpenTelemetry
- Build a full observability stack (metrics, logs, traces)
- Compare OSS stack vs Datadog vs Dynatrace vs AppDynamics
- Automate everything using Ansible and GitHub Actions
